# Battleship
Це гра про два ворогуючі морські флоти, які зустрілись у відкритому морі. Ставай на сторону одного з них у ролі адмірала, сформуй стрій своїх кораблів та розтрощи супротивника.
Правила гри:
Гра стартує з фази розстановки флоту. Кожен корабель має знаходитися на ігровому полі і не торкатися інших кораблів. Курсор миші завжди відповідає верхньому лівому кутку корабля. Ліва кнопка миші, щоби розмістити корабель, права кнопка миші, щоби змінити орієнтацію корабля.
